Mémos formats

Ecrit par : lucas | Date : 05 Dec 2015

La commande ** file** (dans une terminal sur ubuntu) permet de déterminer rapidement le format de certains fichiers quand on ne dispose pas d’indication comme un suffixe pour le savoir : sas ~ $ file note.png note.png: PNG image data, 32 x 32, 4-bit colormap, non-interlaced sas ~tuteurs/tex/documents/doc/doc-2003 $ file questionnaire.ps questionnaire.ps: PostScript document text conforming at level 2.0
